# ayuda-solidaria-rd-venezuela
Plataforma ciudadana solidaria nacida en R.D. para coordinar la ayuda humanitaria hacia Venezuela tras los sismos de junio 2026. Esta herramienta PWA, funcional sin internet, permite consultar insumos urgentes y localizar centros de acopio seguros. Un espacio transparente y colaborativo para gestionar la logística de ayuda de forma eficiente.
